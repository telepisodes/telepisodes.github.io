Welcome to the official telepisodes API documentation

Telepisodes API was conceptualized to provide a better, safer and faster way in contributing link contents in the website. It is accessible for users of the site who wishes to contribute content to the site.

Endpoint

https://telepisodes.org/api/submit

Method

GET

Parameters

  • key - [api key, request this key by sending an email to suggest@telepisodes.net with reason for using the API]
  • t - TVMaze ID (e.g. tvm1234) or TheMovieDB Id (e.g. tmd1234)
  • s - Season (integer)
  • e - Episode (integer)
  • u - URL / Video link to submit

Note

The site is protected by a 3rd party provider and rate limiting is in place. Do not ABUSE the API. Telepisodes implements a one strike policy where abusers are permanently banned from the site if identified as abusing the API or violating any guidelines of the site.
